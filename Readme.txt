Le programme demande à l'utilisateur d'entrer un nombre et ensuite il lui dit si le nombre est multiple
de 2 ou multiple de 3.
Nous avons créer deux fonctions. La première véfirifie si le nombre est multiple de 2 et la deuxieme si le nombre est multiple de 3.

Pour chaque valeur entrée par l'utilisateur, les deux conditions sont véridfiées et le résultat de la vérification est affiché.
